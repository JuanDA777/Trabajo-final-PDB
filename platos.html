Consulta 2

db.menu.find({ categoria: { $gt: 'Entradas' } });

Descripción: Encuentra platos en categorías que vienen después de "Entradas".

Consulta 3

db.menu.find({ tiempo_preparacion: { $gte: 25 } });

Descripción: Busca platos que requieren 25 minutos o más para prepararse.

Consulta 4

db.menu.find({ disponibilidad: { $lt: 15 } });

Descripción: Encuentra platos que tienen menos de 15 unidades disponibles.

Consulta 5

db.menu.find({ popularidad: { $lte: 80 } });

Descripción: Busca platos con un nivel de popularidad de 80 o menos.

Consulta 6

db.menu.find({ nivel_picante: { $ne: 'Suave' } });

Descripción: Encuentra platos que no sean "Suaves" en nivel de picante.

Consulta 7

db.menu.find({ calorias: { $in: [200, 300, 400] } });

Descripción: Busca platos que tengan 200, 300 o 400 calorías.

Consulta 8

db.menu.find({ calorias: { $nin: [350, 700] } });

Descripción: Encuentra platos que no tengan 350 o 700 calorías.

Consulta 9

db.menu.find({ precio: { $gt: 20000 }, disponibilidad: { $gt: 10 } });

Descripción: Busca platos que cuesten más de 20,000 y tengan más de 10 unidades disponibles.

Consulta 10

db.menu.updateOne({ nombre: 'Hamburguesa Clásica' }, { $set: { precio: 22000 } });
